Crie um componente para exibir o ranking dos grupos de receita mais caros no Dashboard.

API: https://seu-dominio.replit.app

ENDPOINT:
GET /api/sinistralidade/grupos-receita/ranking?dataInicio=01/11/2024&dataFim=30/11/2024&limit=5&contratos=263,2444

PARÂMETROS:
- dataInicio (obrigatório, DD/MM/YYYY)
- dataFim (obrigatório, DD/MM/YYYY)
- limit (opcional, default 10, max 100)
- contratos (opcional, lista de apólices separadas por vírgula)

COMPORTAMENTO:
- contratos presente: filtra apenas pelos contratos especificados
- contratos ausente: retorna dados de TODOS os contratos

RESPOSTA:
{
  "data": [
    { "gruporeceita": "EXAMES", "totalProcedimentos": 1250, "valorTotal": 485230.50, "ticketMedio": 388.18 },
    { "gruporeceita": "CONSULTAS", "totalProcedimentos": 3420, "valorTotal": 342000.00, "ticketMedio": 100.00 }
  ],
  "totais": {
    "totalGeral": 1589730.50,
    "totalProcedimentos": 5847,
    "ticketMedioGeral": 271.89
  },
  "pagination": { "limit": 5, "total": 12 }
}

CAMPOS:
- gruporeceita: Nome do grupo
- totalProcedimentos: Quantidade de procedimentos
- valorTotal: Soma dos valores (ordenado DESC)
- ticketMedio: valorTotal / totalProcedimentos
- totais: Considera TODOS os registros do período (não apenas o limit)

EXIBIÇÃO SUGERIDA:
1. Card com título "Top Grupos de Receita"
2. Seção de totais gerais (Total Geral, Procedimentos, Ticket Médio)
3. Lista rankeada com:
   - Badge de posição (1, 2, 3...)
   - Nome do grupo
   - Valor total (destaque)
   - Barra de progresso relativa ao maior valor
   - Quantidade de procedimentos e ticket médio

CÓDIGO ESSENCIAL (com suporte a filtro por contratos):
```tsx
interface TopGruposReceitaProps {
  dataInicio: string;
  dataFim: string;
  limit?: number;
  contratos?: string[];  // Lista de nºs de apólice
}

export function TopGruposReceita({ dataInicio, dataFim, limit = 10, contratos }: TopGruposReceitaProps) {
  const { data } = useQuery({
    queryKey: ['/api/sinistralidade/grupos-receita/ranking', dataInicio, dataFim, limit, contratos],
    queryFn: () => {
      const params = new URLSearchParams({ dataInicio, dataFim, limit: String(limit) });
      if (contratos?.length) params.append("contratos", contratos.join(","));
      return fetch(`${API}/api/sinistralidade/grupos-receita/ranking?${params}`).then(r => r.json());
    }
  });

  return (
    <>
      {data?.data.map((grupo, index) => (
        <div key={grupo.gruporeceita}>
          <Badge>{index + 1}</Badge>
          <span>{grupo.gruporeceita}</span>
          <span>{formatCurrency(grupo.valorTotal)}</span>
          <Progress value={(grupo.valorTotal / data.data[0].valorTotal) * 100} />
        </div>
      ))}
    </>
  );
}
```

INTEGRAÇÃO COM TopBar/FilterContext:
```tsx
import { useFilters } from '@/contexts/FilterContext';

export function useTopGruposReceita({ limit = 5 } = {}) {
  const { dateRange, selectedEmpresas } = useFilters();
  
  const dataInicio = dateRange?.from ? format(dateRange.from, "dd/MM/yyyy") : "";
  const dataFim = dateRange?.to ? format(dateRange.to, "dd/MM/yyyy") : "";

  return useQuery({
    queryKey: ["top-grupos-receita", dataInicio, dataFim, limit, selectedEmpresas],
    queryFn: () => getTopGruposReceita(dataInicio, dataFim, limit, selectedEmpresas),
    enabled: !!dataInicio && !!dataFim,
  });
}
```
